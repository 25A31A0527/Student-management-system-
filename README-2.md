# Student Management System (C++)

## Introduction
This project is a console-based Student Management System developed in C++.  
It helps users manage student records using file handling and menu-driven operations.

---

## Features
- Add student records
- Display all student records
- Search student records by ID
- Store records permanently using file handling
- Easy menu-driven interface

---

## Technologies Used
- C++
- File Handling
- Object-Oriented Programming (OOP)

---

## How the Program Works
1. User selects an option from the menu.
2. Student details are entered.
3. Data is stored in a text file (`students.txt`).
4. User can display or search records anytime.

---

## File Used
- `students.txt` → Stores all student records permanently.

---

## Menu Options
1. Add Student
2. Display Students
3. Search Student
4. Exit

---

## Advantages
- Simple and easy to use
- Permanent data storage
- Fast student record management
- Beginner-friendly project
